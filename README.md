# usage:
**compiling:** (can change joezarchiver to anything u want)
```bash
g++ -std=c++17 -O3 main.cpp -lpthread -lz -o joezarchiver
```
**to find usage info:**
```bash
./joezarchiver
```
and the commands (how to use) will be printed
